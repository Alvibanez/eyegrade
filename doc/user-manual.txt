
# User Manual
ESC: quit (the current image is not saved, but the previous ones were already saved)
DEL: cancel current image and look again for the exam
SPACE: save current image and look for next exam
Use the mouse to select/unselect student's answers when automatic detection fails

Information Display:
Up-left: exam id (starting in 0, autoincrement, see -s command line argument)
Down-left: model: correct / incorrect / undecided (undecided = more than one option marked)

Command line:

$ cd src/
$ python eyegrade.py --help
